# ZGU Lab Work - Git & GitHub

Небольшой учебный проект для отработки базовых команд Git и работы с GitHub.

## Описание проекта

Приложение на Python, демонстрирующее простые арифметические операции:
- вывод приветственного сообщения;
- сложение двух чисел;
- умножение двух чисел.
